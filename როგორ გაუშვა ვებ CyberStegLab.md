CyberStegLab 

ვებ ვერსია
1. Python Flask სერვისი (colorkey_app)
გახსენი terminal, გადი colorkey_app საქაღალდეში და გაუშვი:
bashpython app.py
სერვერი გაეშვება http://localhost:5050-ზე.

2. C# ASP.NET Core API
გახსენი terminal, გადი CyberStegLabLatest.API საქაღალდეში და გაუშვი:
bashdotnet run
API გაეშვება http://localhost:5243-ზე.

3. Frontend
გახსენი cyFront საქაღალდე VS Code-ში და გაუშვი Live Server — მარჯვენა კლიკი index.html-ზე და "Open with Live Server".

სამივე ერთდროულად უნდა მუშაობდეს.
